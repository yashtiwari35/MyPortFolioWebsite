Overview

This project focuses on analyzing sales order data to uncover meaningful insights and support data-driven decision-making. It involves data cleaning, transformation, and visualization to evaluate business performance across different regions, categories, and time periods.

Objective

The primary objective is to understand sales trends, identify high-performing products and regions, and highlight areas where profitability can be improved. The findings are meant to guide strategic business decisions.

Tools & Technologies

  • Python (Pandas, NumPy) — data cleaning and transformation
  • Matplotlib & Seaborn — exploratory data visualization
  • Jupyter Notebook — interactive analysis environment
  • Power BI — interactive dashboard visualization

Key Analysis Performed

  • Data Cleaning and Preprocessing (handling missing values, formatting columns)
  • Sales, Profit, and Order Analysis
  • Region-wise Performance Analysis
  • Category and Sub-category Profitability Analysis
  • Top-selling Products Identification
  • Time-based Trend Analysis (Year-wise sales growth)

Key Insights

  • Technology category generated the highest profit among all categories
  • Furniture category showed comparatively lower profitability
  • Sales increased steadily over time, indicating consistent business growth
  • Certain regions significantly outperformed others in sales contribution
  • High shipping costs and deep discounts impacted profit margins in some cases

Conclusion

This project demonstrates how raw sales data can be transformed into actionable insights using modern data analysis techniques. The findings help businesses make informed decisions to improve sales performance, optimize pricing strategies, and enhance overall profitability.
